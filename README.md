# sudokubot

Browser bot that solves online sudoku puzzles.

Beat the "evil" level on https://websudoku.com every time!

----

### Requirements

- Python 3
- Selenium WebDriver Python bindings:

```
    pip install selenium
```

### Instructions

```
    python3 sudokubot.py
```
