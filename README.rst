=========
sudokubot
=========

selenium webdriver browser-bot that solves sudoku puzzles.

------------
Requirements
------------

 * Python 2.6+
 * Selenium WebDriver client bindings::
    
    pip install selenium
    
------------
Instructions
------------

pure selenium webdriver version::

    $ python sudokubot.py

SST framework version::

    $ ./sst-run sudokubot-sst.py
